# Issue tracker: GitHub

Issues and PRDs for LifeLedger live in GitHub Issues at `tmcuh1982/LifeLedger`.

Use the `gh` CLI from the repository root:

- Create: `gh issue create --title "..." --body "..."`
- Read: `gh issue view <number> --comments`
- List: `gh issue list --state open`
- Edit labels: `gh issue edit <number> --add-label "..."`
- Close: `gh issue close <number> --comment "..."`

When a workflow asks to publish work to the issue tracker, create a GitHub issue.
